# ByteJam

A bytebeat-inspired expression-based social network.

Create and share musical expressions.

## Features

- [x] User registration and login
- [x] Manage profile (edit username, delete account)
- [x] Create and share musical expressions (share link)
- [x] Create posts
- [x] Edit posts
- [x] Explore posts (recent, popular)
- [x] Add to favorites
- [x] Transport controls
- [x] Visualizer
- [ ] Master gain
- [ ] Fork
- [ ] Sandboxing
- [ ] Follow users
- [ ] Report posts or users
- [ ] Tags
- [ ] Personalized feed
- [ ] Search expressions by name and tags
