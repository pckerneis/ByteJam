# Bitebeats

A bytebeat-inspired expression-based social network.

Create and share musical expressions.

## Tech stack

Frontend:

- Vite
- React
- Expo
- Web Audio API

Backend:

- Node.js
- Supabase

## Roadmap

### MVP

- User registration and login
- Create and share musical expressions
- Explore expressions
- Like expressions and save favorites

### Polish and security

- Rate limiting
- Report expressions or users
- Sandboxing
- Admin panel
- Analytics
- Terms of service
- Privacy policy

### More features

- User description
- Tag
- Feed
- Search expressions by name and tags
- Follow users and see their expressions
